Добро пожаловать!

Для заучивания слов в нашем приложении применяется метод интервальных повторений.
Интервальные повторения — это метод эффективного запоминания, его суть в том, чтобы повторять заученную информацию через определенные промежутки времени (без повторений она быстро забудется)
Для того, чтобы мы могли подобрать индивидуальные интервалы для каждого изучаемого слова, предлагаем Вам для начала выбрать подходящие Вам настройки.
